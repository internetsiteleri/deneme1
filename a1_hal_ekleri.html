Hal Ekleri (Kelishik Qo‘shimchalari)

A1 / A2 Seviyesi  ·  Yeni Başlayanlar İçin
💡 Türkçede isimler cümle içinde farklı görevler üstlenir. İsmin sonuna eklenen hâl ekleri (durum ekleri) sayesinde kelimenin cümledeki işlevi belirlenir. Aşağıdaki tabloda her bir hâl ekinin ekleri, soruları, Türkçe ve Özbekçe örnekleri gösterilmiştir.
1. Yalın Hâl 📘 Bosh kelishik (ek yo‘q)
❓ Kim?   Ne?
➕ EK YOK
∅ (ek yok)

İsim hiçbir durum eki almaz, yalın biçimde kullanılır.

Kitap burada.
📖 Kitob bu yerda.
Ali öğrencidir.
👨‍🎓 Ali talaba.
2. -İ Hâli (Belirtme) 🎯 Tushum kelishigi
❓ Neyi?   Kimi?
➕ -ı / -i / -u / -ü
-yı -yi -yu -yü
📌 Ünlü uyumuna göre ek değişir. Ayrıca 📌 Kelime ünlü harfle bitiyorsa araya -y sesi girer. (örn: araba → arabayı, elbise → elbiseyi)

Belirtilen nesneyi, doğrudan etkilenen varlığı gösterir.

Kitabı okudum.
Kitobni o‘qidim.
Çantayı aldım.
Sumkani oldim.
Arabayı yıkadım (araba + yı).
Mashinani yuvdim.
3. -E Hâli (Yönelme) 🧭 Yo‘nalish kelishigi
❓ Nereye?   Kime?
➕ -a / -e
-ya -ye
📌 Ünlü uyumuna göre ek değişir. 📌 Kelime ünlü harfle bitiyorsa araya -y sesi girer. (örn: araba → arabaya, bahçe → bahçeye)

Yönelme bildirir. Kalın/ince ünlü uyumu.

Okula gittim.
Maktabga kettim.
Arabaya bindim.
Mashinaga mindim.
Eve döndüm.
Uyga qaytdim.
4. -DE Hâli (Bulunma) 📍 O‘rin-payt kelishigi
❓ Nerede?   Kimde?
➕ -da / -de / -ta / -te
📌 Sert ünsüzlerden sonra -ta/-te gelir. (f, s, t, k, ç, ş, h, p)

Bulunma anlamı bildirir.

Evde bekliyorum.
Uyda kutyabman.
Okulda ders var.
Maktabda dars bor.
Saat ikide buluşalım.
Soat ikkda uchrashaylik.
5. -DEN Hâli (Ayrılma) 🚀 Chiqish kelishigi
❓ Nereden?   Kimden?
➕ -dan / -den / -tan / -ten
📌 Sert ünsüzlerden sonra -tan/-ten gelir. (ç, f, h, k, p, s, ş, t)

Çıkma, ayrılma, uzaklaşma anlamı bildirir. Kaynak veya başlangıç noktası.

Arabadan in.
Mashinadan tush.
İstanbul‘dan geliyorum.
Istanbuldan kelyapman.
Arkadaşımdan mektup aldım.
Do‘stimdan xat oldim.
✨ Hatırlatma / Eslatma: Türkçedeki hal ekleri ünlü uyumuna (kalın/ince, düz/yuvarlak) ve ünsüz benzeşmesine göre değişir.
📌 -İ hali ve -E hali için: Ünlüyle biten kelimelere eklenirken yardımcı -y- ünsüzü girer (araba → arabayı, araba → arabaya).
📌 -DE ve -DEN hali: Sert ünsüzle biten kelimelerde -ta/-te / -tan/-ten kullanılır (kitap → kitapta, kitaptan).
📖 Örnek (1): Yalın → kitap / kitob    -İ hali → kitabı / kitobni    -E hali → kitaba / kitobga    -DE hali → kitapta / kitobda    -DEN hali → kitaptan / kitobdan
